Features

主な機能

瞬間英作文

日本語からテンポよく英文を組み立てる練習。並べ替えモードと入力モードの 2 段階で着実にステップアップできます。

AI添削

意味・自然さ・文法の 3 観点で AI が即座にフィードバック。「なぜこの表現が良いのか」も学べます。

単語学習

例文穴埋め形式で、意味だけでなく使い方まで身につく。レベル別に整理された語彙データベースを内蔵。

発音フィードバック

音声解析エンジンによる発音採点。AI が改善ポイントを具体的にアドバイスし、ネイティブに近い発音へ。

進捗の見える化

日々の学習量・正答率・継続日数をダッシュボードで確認。週間目標を設定して習慣化を後押しします。

マルチデバイス同期

iPhone と iPad で学習履歴をリアルタイム同期。Apple ID または Google アカウントでログインするだけ。

FAQ

よくある質問

サインインしないと使えませんか？

サインインなしで基本機能をご利用いただけます。複数デバイスで学習履歴を同期したい場合のみ、Apple ID または Google アカウントでサインインしてください。

無料体験期間中に解約したら課金されますか？

無料体験期間（7 日間）の終了前に解約すれば、料金は一切発生しません。Apple ID のサブスクリプション設定からいつでも解約できます。

動作環境は？

iOS 18.0 以降の iPhone と iPad に対応しています。Apple Silicon 搭載 Mac でも iPad 互換モードでご利用いただける予定です。

個人情報の取り扱いは？

メールアドレスとユーザー ID、学習進捗、購入履歴のみを取得し、トラッキングは一切行いません。詳細は プライバシーポリシー をご確認ください。
